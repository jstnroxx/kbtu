Album Browser — is a web-applictaion which uses Angular's Routing, HTTP & Services.
It fetches data from a public REST API, implements navigation between multiple views, and creates a service layer to manage data.
The focus is on routing configuration, HTTP requests, Observables, and building a multi-page single-page application (SPA).
